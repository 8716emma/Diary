
# 돌림판 웹 애플리케이션

## 개요

이 애플리케이션은 사용자가 직접 옵션을 추가하고 돌릴 수 있는 인터랙티브한 돌림판 웹사이트입니다. 모던 웹 기술을 사용하여 제작되며, 아름다운 디자인과 뛰어난 사용자 경험을 제공하는 것을 목표로 합니다.

## 기능

*   **옵션 추가:** 사용자는 텍스트 입력 필드를 통해 돌림판에 원하는 옵션을 추가할 수 있습니다.
*   **돌림판 돌리기:** '돌리기' 버튼을 클릭하면 돌림판이 회전하며, 서서히 멈추어 하나의 옵션을 선택합니다.
*   **결과 표시:** 돌림판이 멈추면 선택된 옵션이 화면에 표시됩니다.
*   **반응형 디자인:** 웹과 모바일 등 다양한 화면 크기에서 최적화된 화면을 보여줍니다.
*   **시각적 디자인:** 현대적인 색상, 폰트, 애니메이션 효과를 사용하여 시각적 즐거움을 더합니다.

## 현재 요청 계획

1.  **`index.html` 수정:**
    *   페이지 제목을 "돌림판 웹"으로 변경합니다.
    *   돌림판을 그릴 `<canvas>` 요소를 추가합니다.
    *   옵션 추가를 위한 `<input>`과 `<button>`을 추가합니다.
    *   돌림판을 돌리기 위한 `<button>`을 추가합니다.
    *   결과를 표시할 요소를 추가합니다.

2.  **`style.css` 수정:**
    *   돌림판, 포인터, 입력 필드, 버튼 등 전체적인 스타일을 디자인 가이드라인에 맞게 적용합니다.
    *   화면 중앙에 돌림판이 위치하도록 레이아웃을 조정합니다.

3.  **`main.js` 수정:**
    *   Canvas API를 사용하여 돌림판과 각 옵션을 그리는 로직을 구현합니다.
    *   사용자가 옵션을 추가하고 돌림판을 돌리는 것에 대한 이벤트 리스너를 추가합니다.
    *   `requestAnimationFrame`을 사용하여 부드러운 돌림판 회전 애니메이션을 구현합니다.
    *   회전이 멈춘 후 선택된 옵션을 계산하고 결과를 표시하는 로직을 작성합니다.
